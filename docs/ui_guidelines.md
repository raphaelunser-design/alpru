# UI Guidelines

## Brand
- Name: Alpivo
- Primary color: #0f172a
- Logo/icon: /public/brand

## Layout & spacing
- Use consistent card padding (p-5/p-6)
- Prefer 2-3 column grids on desktop, 1 column on mobile
- Avoid dense layouts; add vertical breathing room

## Components
- Cards: subtle lift on hover, soft shadow
- Buttons: rounded-xl, clear hierarchy (primary/secondary)
- Empty states: helpful, friendly, with a clear next action

## Imagery
- Resort cards use Next/Image and show a local fallback
- External resort images should be optimized via remotePatterns

## Motion
- Short page transitions (<= 200ms)
- Respect prefers-reduced-motion
